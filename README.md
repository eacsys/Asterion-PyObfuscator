## 📋 Prerequisites

* Python 3.x
* Windows, Linux, or macOS

---

## 💻 Installation

1. Clone the repository:

```bash
git clone https://github.com/eacsys/Asterion-PyObfuscator.git
cd Asterion-PyObfuscator
```

---

## 🚀 Usage

1. Place the Python file you want to obfuscate in the project directory.
2. Run the obfuscator:

```bash
python Asterionv2.py your_file.py
```
